Functions

  • add_circleAdd Student
    This function is to add a new student. When you click on "Add Student" on the nav bar it will take you to the new page to add a student. You can add a student with the given fields on the page.

    It will not let you add a student with same student id of same email.

  • contactsAll Students
    All students page consists of all the students which are added now. Once you add all the students using add student function. You can see all the students on "All Student" page.

    You can edit the name of the students using edit button. This will not let you change the student ID of student Email. You can just change the first name and last name of the student.

    You can also delete any student by pressing delete button. This will delete that student form the database.
  • descriptionEssay Grader
    Essay Grader Helps you to grade students. This takes two thing. One is it takes essay of student and essay of proffessor. This gives the total spelling mistakes, Grammetical mistakes and it compares the essay to the proffessor's essay to see how much it matches.
